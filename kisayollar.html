Anasayfa Hakkımda Kısayollar İletişim(Yeni Sekmede Aç)

CTRL + SHIFT + P ile Vscode menuyu açabilirsiniz. Daha sonra keyboard shortcuts yazın.
İçerisinde istediğiniz klavye kısayollarını belirleyebilirsiniz.

  • CTRL + C : kopyalama (copy)
  • CTRL + Z : geri al (undo)
  • CTRL + X : kesmek (cut)
  • CTRL + F : mevcut sayfada bulmak (find)
  • CTRL + SHIFT + F : bütün projede bulmak (find)
  • CTRL + D : mevcut satırı aynen alta kopyalama (copy line down)
  • ALT + TAB : pencere değişmek
  • Format Document (Kodu otomatik formatlama yani düzenleme için) CTRL + SHIFT + I ayarlanabilir.
